金沢学院大学 経済情報学部「ネットワーク科学と経済Ⅱ」（学部4年次配当・担当: 後藤弘光）の演習教材リポジトリです。

> 前提科目: [ネットワーク科学と経済Ⅰ](https://github.com/hgphys/ns1)

---

## 授業概要

「ネットワーク科学と経済Ⅰ」で学んだグラフ理論の基礎の上に、リンク予測・コミュニティ抽出・ネットワーク構造の統計モデル・伝播現象など発展的なネットワーク分析手法を Python（[NetworkX](https://networkx.org/)）で実践する。社会経済ネットワークの実データを題材に、構造の予測やコミュニティ抽出を通じて将来の社会構造や様々な伝播現象について考察する。

## 到達目標

- ネットワークの構造予測の手法について理解し、社会経済ネットワークの実データに応用できる
- コミュニティを抽出する手法について理解し、社会経済ネットワークの実データに応用できる
- ネットワーク科学の視点から将来の社会構造の予測や様々な伝播現象について考察できる

## 環境構築（初回のみ）

> **「ネットワーク科学と経済Ⅰ」で `ns_env` を作成済みの場合、この手順は不要です。**

### 1. Anaconda のインストール

以下のサイトを参考にインストールしてください。

- [Python 環境構築ガイド](https://www.python.jp/install/install.html)
- [Anaconda](https://www.anaconda.com/products/distribution)

> ユーザ名が日本語の場合、インストール先のパスに注意してください。

### 2. 仮想環境の作成

```bash
conda create -n ns_env python=3.11 -y
```

### 3. 仮想環境の有効化

```bash
conda activate ns_env
```

プロンプトに `(ns_env)` と表示されれば OK です。

### 4. 必要パッケージのインストール

```bash
pip install -r requirements.txt
```

## 課題の取り組み方

### 前半a（ns2a_02〜ns2a_04）

各回のフォルダには以下のファイルが含まれます。

```
ns2a_XX/
├── ns2a_XX.ipynb       # 課題本体（Jupyter Notebook）
├── ns2a_XX_01.py       # 問題1用スクリプト
├── ns2a_XX_02.py       # 問題2用スクリプト
├── ns2a_XX_03.py       # 問題3用スクリプト
└── data/               # データファイル（ある場合）
```

> `data/`, `figs/` など空のフォルダには `.gitkeep` という空ファイルが入っています。これは Git で空フォルダを保持するための目印です。無視して構いません。

ノートブック内で `.py` ファイルは以下のように実行できます。

```python
%run ns2a_XX_01.py
```

### 前半a 実践演習（ns2a_05-07）

テンプレート `ns2a_05-07_学籍番号.ipynb` をコピーし、ファイル名の「学籍番号」を自分の学籍番号に変更してから取り組んでください。

> **個別データの配布について**
> 実践演習では、受講生ごとに異なるネットワークデータを使用します。
> Moodle コースから自分の学籍番号のデータファイルをダウンロードし、`ns2a_05-07/data/` フォルダに保存してから課題に取り組んでください。
> このリポジトリには配布用データは含まれていません（受講生のみ Moodle 経由で取得）。

### 手順

1. **仮想環境を有効化する**

    ```bash
    conda activate ns_env
    ```

2. **JupyterLab を起動する**

    ```bash
    jupyter lab
    ```

3. **課題に取り組む** — 該当回のノートブックを開き、指示に従います。

4. **回答を提出する** — Moodle コースの小テストに回答します。

5. **作業を終えたら無効化する**

    ```bash
    conda deactivate
    ```
